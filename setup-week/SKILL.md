## Setup Week

Scaffolds this week's periodic notes (Monday through Friday daily notes and the weekly summary note) using Claude's Write tool. Skips any notes that already exist. After scaffolding, reviews the weekly note and asks for weekly priorities.

### Steps

1. Calculate the dates for this week:
   - Determine today's date from the system context (`currentDate`)
   - Find Monday of the current week (if today is Sunday, use the coming Monday)
   - Derive Monday through Friday dates (YYYY-MM-DD format)
   - Calculate the ISO week number (YYYY-Www format, e.g. `2026-W13`)
   - Calculate previous and next week strings for navigation links

2. For each of the 6 notes to create (1 weekly + 5 daily), check if the file already exists using the Read tool. If it exists, skip it and note "Exists". If it does not exist, create it with the Write tool using the exact templates below.

   **Weekly note** — path: `Weekly/YYYY-Www.md`
   ```
   ---
   type: weekly
   week: "YYYY-Www"
   date_start: YYYY-MM-DD
   date_end: YYYY-MM-DD
   ---

   # YYYY-Www

   << [[PREV-WEEK|← Previous Week]] | [[NEXT-WEEK|Next Week →]] >>

   # Weekly Priorities
   1.
   2.
   3.

   # Rollup
   <!-- Generated by /weekly-rollup command -->

   # Insights & Patterns

   # Open Action Items
   ```dataview
   TASK
   WHERE !completed AND file.folder = "Daily"
   SORT file.name DESC
   LIMIT 20
   ```

   # Notes Created This Week
   ```dataview
   LIST
   WHERE file.cday >= date(WEEK-START) AND file.cday <= date(WEEK-END)
   SORT file.cday ASC
   ```

   # Reflection

   ---

   # Daily Notes
   - [[MON-DATE|Monday MON-DATE]]
   - [[TUE-DATE|Tuesday TUE-DATE]]
   - [[WED-DATE|Wednesday WED-DATE]]
   - [[THU-DATE|Thursday THU-DATE]]
   - [[FRI-DATE|Friday FRI-DATE]]
   ```

   **Daily note** — path: `Daily/YYYY-MM-DD.md`
   ```
   ---
   type: daily
   date: YYYY-MM-DD
   week: "[[YYYY-Www]]"
   ---

   # DayName, YYYY-MM-DD

   << [[PREV-DATE|← Previous]] | [[YYYY-Www|Week]] | [[NEXT-DATE|Next →]] >>

   # Scratchpad

   # Reflection
   ```

3. After creating/checking all notes, report what happened — e.g. "Created: Weekly/2026-W13.md, Daily/2026-03-23.md through Daily/2026-03-27.md. Skipped: Daily/2026-03-24.md (already existed)."

4. Read the weekly note.

**Incorrect:**

Creating the files and saying "Done" without reading the weekly note or asking about priorities. The scaffolding is just setup — the value is the priority-setting conversation.

**Correct:**

Computing the week dates, creating any missing notes, reporting "Created daily notes for Mon–Fri and weekly note 2026-W13. Wednesday already existed so it was skipped."
